```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Deploy files on Firebase Hosting
```
npm run deploy
```
or

```
firebase deploy --only hosting
```
